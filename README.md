## üßÆ About the N-Queens Problem
The **N-Queens Problem** asks:
> How can we place N queens on an N√óN chessboard so that no two queens attack each other (no same row, column, or diagonal)?

This program uses a **backtracking approach** ‚Äî a recursive search that explores possible placements, backtracks when conflicts occur, and continues until all solutions are found.
